## La marca personal — Concepto

**Posicionamiento:** "No vendo propiedades. Construyo sistemas que las venden."

No es un broker tradicional ni un dev genérico. Es el operador que construye la infraestructura digital que las inmobiliarias no saben hacer. Cada propiedad listada en el sitio es una demostración en vivo de sus sistemas.

**Tres capas del sitio:**
1. **Propiedades activas** — Cada listing es un demo de los sistemas funcionando (landing, WhatsApp IA, contenido, drone, tour 360°)
2. **Sistemas como servicio** — Landing por propiedad, WhatsApp IA, contenido & reels, sistema completo de ventas
3. **Track record + desarrollo** — Historia (Flaternity → Colombia), métricas reales, y pipeline de desarrollo en Bogotá

## Marco legal — Corretaje en Colombia

- **No se requiere licencia** para intermediar ventas (Código de Comercio, Arts. 1340-1346)
- **Sí se requiere:** Matrícula mercantil (Cámara de Comercio), RUT con CIIU 6820, factura electrónica si responsable de IVA
- **Comisiones estándar:** 3% urbano, 4-5% rural/campestre, negociable en alto valor
- **Impuestos sobre comisiones:** IVA 19% (si responsable), retención en la fuente 10-11%
- **Contrato:** Contrato de corretaje (Arts. 1340-1346 C.Co.) — escrito recomendado con % comisión, exclusividad, y timing de pago
- **Legislación pendiente:** Proyecto de Ley 242/2025 busca profesionalizar el rol con registro obligatorio

## Stack técnico — Sitio web

**Recomendado:**
- **Framework:** Next.js App Router
- **CMS:** Payload CMS (MIT, se instala dentro del proyecto Next.js, code-first TypeScript)
- **DB:** PostgreSQL via Drizzle ORM (Supabase o Neon, ~$5-15/mes)
- **Imágenes:** Cloudinary (free tier 25 credits/mes) o Payload media + S3 + next/image
- **Mapas:** Mapbox (50K loads/mes gratis, mejor customización que Google Maps)
- **Búsqueda:** PostgreSQL queries via Payload Local API inicialmente; Meilisearch cuando escale
- **WhatsApp:** Click-to-chat links inicialmente → WhatsApp Business API (Meta Cloud API) cuando crezca volumen
- **SEO:** generateMetadata(), sitemap.ts, JSON-LD (RealEstateListing, GeoCoordinates, RealEstateAgent), URLs geográficas
- **Hosting:** Vercel Pro ($20/mes) para frontend + Hetzner/DO VPS (~€10-20/mes) para Payload + PostgreSQL
- **Costo total infraestructura:** $35-75/mes
- **Dominio:** .com principal + .com.co secundario + .co reservado
